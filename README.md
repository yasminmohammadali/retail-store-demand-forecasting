<h1 align="center">Retail Store Demand Forecasting Dashboard (Excel + Tableau)</h1>

<p align="center">
Retail sales analytics and demand forecasting dashboards created using the same dataset in <b>Microsoft Excel</b> and <b>Tableau</b>.
</p>

<hr>

<h2>📌 Project Overview</h2>
<p>
This project focuses on analyzing retail store sales data to perform <b>demand forecasting</b> and uncover business insights related to
sales trends, seasonality, inventory levels, discounts, product categories, and regional performance.
</p>

<p>
Two separate interactive dashboards were built using the <b>same dataset</b>:
</p>
<ul>
  <li>Dashboard 1 – Built in <b>Microsoft Excel</b></li>
  <li>Dashboard 2 – Built in <b>Tableau</b></li>
</ul>

<p>
This project demonstrates how the same business problem can be solved using different analytics and visualization tools.
</p>

<hr>

<h2>🎯 Objectives</h2>
<ul>
  <li>Forecast product demand based on historical sales</li>
  <li>Analyze monthly and seasonal sales patterns</li>
  <li>Study inventory behavior and discount impact</li>
  <li>Compare sales performance across regions</li>
  <li>Identify high-demand product categories</li>
</ul>

<hr>

<h2>📊 Key KPIs</h2>
<ul>
  <li>Total Revenue</li>
  <li>Total Units Sold</li>
  <li>Average Inventory Level</li>
  <li>Average Discount</li>
  <li>Average Price</li>
</ul>

<hr>

<h2>📈 Dashboard Insights</h2>
<ul>
  <li>Seasonality plays a major role in product demand</li>
  <li>Certain regions consistently outperform others in sales</li>
  <li>Groceries and Clothing show higher demand compared to other categories</li>
  <li>Inventory levels fluctuate based on forecasted demand</li>
  <li>Discount strategies directly influence unit sales</li>
</ul>

<hr>

<h2>🛠 Tools Used</h2>
<ul>
  <li>Microsoft Excel (Pivot Tables, Charts, Forecast Sheet, Slicers)</li>
  <li>Tableau (Interactive Dashboards, Filters, Forecasting)</li>
</ul>
